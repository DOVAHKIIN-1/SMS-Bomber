اول از همه نیازه که مخازن ترموکس رو آپدیت کنیم و اجازه دسترسی ها رو به برنامه بدیم برای این کار کد های زیر را به ترتیب وارد کنید:
apt update

apt upgrade

apt install python -y

apt install python2 -y

apt install python3 -y

apt install git -y

termux-setup-storage
 بعد از وارد کردن این کد حتما اجازه دسترسی رو به برنامه بدید🔼

خب حالا ابزار رو درجایی که استخراج کردیم اجرا می‌کنیم (پیشنهاد میشه که فایل رو در محل ذخیره دستگاه یا همون حافظه اصلی استخراج کنید) و بعد با زدن این کد پوشه رو فرا می‌خونیم: 
cd /sdcard/SMS-Bomber

و بعد کدهای زیر رو به ترتیب وارد میکنیم:

python install.py

python Sms-Bomber.py

اکنون شماره تلفن مورد نظر را وارد کنید

بعد بین دو گزینه {0 و 1} یکی را وارد کنید (پیشنهاد میشه عدد یک رو وارد کنید) و تمام

برای متوقف کردن ابزار کلید های (c و Ctrl) را به ترتیب وارد کنید
